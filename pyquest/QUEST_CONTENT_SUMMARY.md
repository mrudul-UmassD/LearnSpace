## ðŸ”§ Technical Details

### JSON Schema:
- **id:** Unique identifier (lowercase-with-hyphens)
- **world:** World identifier (python-basics, data-structures)
- **title:** Quest display name
- **story:** Background narrative
- **instructions:** Step-by-step guidance
- **starterCode:** Initial code template
- **solutionHidden:** Reference solution
- **tests:** Array of test objects
- **hints:** Array of 3 hint objects
- **xpReward:** Integer (1-1000)
- **hintUnlockAttempts:** Integer (1-10)
- **difficulty:** Enum (beginner/intermediate/advanced)
- **order:** Sequential position

### Test Types Supported:
- `output`: Exact output matching
- `variable_exists`: Variable defined
- `variable_type`: Type checking
- `variable_value`: Value matching
- `function_call`: Function exists and callable
- `list_contains`: Item in list
- `list_length`: List size
